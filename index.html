REST API

All endpoints are open — no auth required. Base URL:

GET /api/files List all items
curl /api/files
POST /api/upload Upload a file (simple, <5MB recommended)
curl -F 'file=@screenshot.png' -F 'ttl=7d' /api/upload
POST /api/upload/init Initialize chunked upload (for large files)
curl -X POST -H 'Content-Type: application/json' \ -d '{"filename":"big.zip","size":524288000,"mime_type":"application/zip","ttl":"7d"}' \ /api/upload/init
POST /api/upload/chunk Upload a chunk (multipart: upload_id, chunk_index, chunk)
curl -F 'upload_id=abc123' -F 'chunk_index=0' -F 'chunk=@chunk0.bin' /api/upload/chunk
GET /api/upload/status/{upload_id} Check which chunks have been received (for resume)
curl /api/upload/status/{upload_id}
POST /api/upload/complete Finalize chunked upload (reassembles file)
curl -X POST -H 'Content-Type: application/json' \ -d '{"upload_id":"abc123"}' \ /api/upload/complete
POST /api/text Create a text snippet
curl -X POST -H 'Content-Type: application/json' \ -d '{"content":"hello world","name":"note.txt","ttl":"7d"}' \ /api/text
GET /api/files/{id} Download a file or get text content
curl /api/files/{id} -o file.png
GET /api/text/{id} Get raw text snippet
curl /api/text/{id}
DELETE /api/files/{id} Delete an item
curl -X DELETE /api/files/{id}
GET /f/{id} Direct file access (in browser)
/f/{id}
GET /t/{id} Direct text access (raw, in browser)
/t/{id}
GET /health Health check
curl /health

Available Tools

list_files
List all pasted files and text snippets with metadata (id, name, type, size, created, expires, url).
Parameters: (none)
get_file
Get the content of a file or text snippet by ID. Returns base64-encoded content for files, plain text for text snippets.
Parameters: id (string, required) — The unique ID of the item
get_file_url
Get the public URL for a file or text snippet by ID.
Parameters: id (string, required) — The unique ID of the item
upload_file
Upload a file (base64-encoded content + filename). Returns the item ID and URL.
Parameters: filename (string, required), content (string, required, base64), mime_type (string, optional), ttl (string, optional: 1h, 1d, 7d, 30d, never)
create_text
Create a text snippet. Returns the item ID and URL.
Parameters: content (string, required), name (string, optional), ttl (string, optional: 1h, 1d, 7d, 30d, never)
get_text
Get the raw text content of a text snippet by ID.
Parameters: id (string, required) — The unique ID of the text snippet
delete_file
Delete a file or text snippet by ID.
Parameters: id (string, required) — The unique ID of the item to delete
